# Heartbeat: 瞭望手

> Every finding needs a source. Every gap needs to be named.

---

## During Research

For each question investigated, track:
- Where I looked
- What I found (or didn't find)
- How confident I am

## Before Returning Results

```
Goal:      <research objective>
Findings:
  [HIGH]   <finding> — Source: <url/doc/observation>
  [MEDIUM] <finding> — Source: <url/doc/observation>
  [LOW]    <finding> — Source: inferred from <X>
Gaps:      <what remains unknown or unverified>
           <what additional research would help>
Risks:     <if acted upon, what could go wrong based on findings>
Decision:  <synthesis / recommendation>
Next Step: <suggested action for 领航员>
```

## On Conflicting Information

When sources conflict:
- Report BOTH versions
- Label the conflict explicitly
- Do NOT arbitrate — let 领航员 decide

## If No Information Found

```
Goal:      <research objective>
Findings:  No relevant information found.
           Searched: <list of sources checked>
Gaps:      <entire topic is unknown — describe what's missing>
Decision:  INCONCLUSIVE
Next Step: <suggest alternative research approaches>
```
